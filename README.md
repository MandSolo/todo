## Functionality
- Add a new task
- Mark as completed and filter by active/completed
- Delete tasks

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn start
```

### Compiles and minifies for production

```
yarn build
```

### Testing

```
yarn test
```

### Linting

```
yarn lint
```
```
yarn stylelint
```
